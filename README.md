# dotfiles 文件管理

## Features
-  采用[rcm](https://github.com/thoughtbot/rcm) 来管理
- [alacritty](https://github.com/alacritty/alacritty),  [tmux](https://github.com/tmux/tmux/wiki), [helix](https://github.com/helix-editor/helix) 统一样式,简洁透明
- 使用[zimfw](https://github.com/zimfw/zimfw) 管理`zsh`,[tpm](https://github.com/tmux-plugins/tpm)管理 [tmux](https://github.com/tmux/tmux)
- [mise](https://github.com/jdx/mise) 统一开发环境和加载 `macos` 加密的环境变量
- 新增[aerospace](https://github.com/nikitabobko/AeroSpace) 管理窗口布局
